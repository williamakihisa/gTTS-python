this is API based web server using python
to run the API you need :
- python version 3 or more
- install service ffmpeg
- install python plugins based on requirements.txt

how to use :
1. setup .env files to adjust the path, port for API web service, and the project name
2. run with python3 textvoiceserve.py
3. there wil be 2 API running in this server, /checkfile to check the files in path, and POST for upload the text to be converted
